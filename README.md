# NAGANO_CAKE

DMM WEBCAMPのチーム開発課題です。
架空の洋菓子店「ながのCAKE」の商品を通販するためのECサイトを作成しました。
### [◆案件の背景]
元々近隣住民が顧客だったが、昨年始めたInstagramから人気となり、全国から注文が来るよう
になった。InstagramのDMやメールで通販の注文を受けていたが、情報管理が煩雑になってきたため、管理機能を含んだ通販サイトを開設しようと思い至った。
